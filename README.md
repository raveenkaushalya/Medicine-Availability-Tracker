# Medicine Availability Tracker

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Overview

The Medicine Availability Tracker is a real-time system designed to help people in Sri Lanka find the availability of medicines across pharmacies and medical stores. This is a 2nd year group project aimed at solving the critical problem of medicine shortages and availability transparency.
